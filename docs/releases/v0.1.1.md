# Pathmark v0.1.1

Public cleanup release after the first npm publish.

## Changes

- Replaced old adapter-specific public wording with neutral legacy-memory language.
- Renamed the migration helper to `npm run import:legacy`.
- Renamed Codex hook takeover flag to `--replace-legacy-hooks`.
- Renamed Codex status field to `legacyHooksPresent`.
- Removed stale ignore and documentation references from the repository and npm package contents.

## Install

```bash
npm install -g pathmark
```
