# Pathmark v0.1.0

Pathmark is a local-first MCP memory layer for AI coding agents.

## Highlights

- Local JSONL memory store at `~/.pathmark/memory/memory.jsonl`.
- MCP tools for saving, searching, retrieving, and deleting memory.
- Cross-harness setup CLI: `pathmark setup <client>`.
- Setup coverage for Codex, Claude Code, Claude Desktop, Cursor, opencode, Gemini CLI, generic MCP clients, command-backed CLIs, and OpenAI-compatible providers such as Kimi, GLM/Z.ai, OpenRouter, and LiteLLM.
- Codex auto-capture adapter with install/status/uninstall commands.
- Legacy import path that preserves the original source store.
- Optional synthesis modes: client, command, Codex CLI, and OpenAI-compatible API.

## Install

```bash
npm install -g pathmark
```

Then generate setup instructions:

```bash
pathmark setup list
pathmark setup codex
pathmark setup claude-code
pathmark setup opencode
pathmark setup gemini-cli
```

## Codex Auto-Capture

```bash
pathmark codex install --replace-legacy-hooks
pathmark codex status
```

`--replace-legacy-hooks` removes old compatible hook commands from Codex config without deleting memory files.

## Notes

This is a first public release. The default path is local-first and API-key-free; hosted sync, encryption, and additional auto-capture adapters are future work.
